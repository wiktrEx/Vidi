# Vidi
Vidi – Your Ultimate YouTube Studio Companion
Vidi is a high-performance desktop application built for YouTube creators who need deeper insights and a cleaner workflow than the standard Creator Studio. Built with Rust and modern web technologies, it focuses on speed, privacy, and actionable analytics.

KEY FEATURES
INTELLIGENT DASHBOARD: A bird's-eye view of your channel's performance, including subscribers, views, and watch time from the last 30 days.

ADVANCED ANALYTICS: Interactive charts showing view trends and subscriber growth, powered directly by the official YouTube Analytics API.

AUDIENCE RETENTION CURVES: Visualize exactly where viewers drop off in your videos with precise retention graphs to help you improve your content.

CONTENT MANAGEMENT: Browse your long-form videos, Shorts, and Live streams in a sleek, dark-themed, and organized interface.

SMART UPLOAD MODULE: Upload videos to YouTube directly from the app while managing titles, descriptions, and visibility settings.

AI-POWERED REPORTS: (Coming Soon) Integration with Claude AI to provide automated summaries and growth tips based on your channel's data.

SECURITY & PRIVACY
Official Google Login: The application uses official Google/YouTube OAuth for secure authentication.

Verification Notice: As Vidi is an independent open-source project, you may see a "Google hasn't verified this app" warning during login. This is normal for new indie projects—your data remains secure and is processed only on your local machine.

No Middleman Servers: Vidi connects directly to Google APIs. Your tokens and analytics never touch any external third-party servers.

GETTING STARTED
Download the latest vidi.exe from the Releases section.

Run the application (If Windows SmartScreen appears, click "More info" -> "Run anyway").

Click "Login with Google" to sync your channel statistics.

TECH STACK
Backend: Rust (High performance and memory safety)

Frontend: HTML5, CSS3 (Modern Dark Mode / Glassmorphism UI)

APIs: YouTube Data API v3 & YouTube Analytics API
